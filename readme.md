# 📸 Camera & Accessory Rental Management System


>A web-based Rental Management System designed for managing camera and accessory rentals efficiently. The platform allows users to browse products, rent items, and perform transactions, while admins can manage inventory and track orders.
---

## ✨ Features

| Feature | Description |
|---|---|
| 👤 **User Authentication** | Secure login & registration system |
| 🛒 **Cart & Rentals** | Browse, add to cart, and rent products |
| 💳 **Checkout & Payments** | UPI QR-based transaction system |
| 📦 **Order Tracking** | Real-time rental order status |
| ⭐ **Feedback System** | User reviews and ratings |
| 🛠️ **Admin Panel** | Manage inventory, orders & users |

---

## 🛠️ Tech Stack

```
Frontend   →   EJS · HTML · CSS
Backend    →   Node.js · Express.js
Database   →   MySQL
```

---

## ⚙️ Getting Started

### Prerequisites
- Node.js (v14+)
- MySQL

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/Rental_website.git

# 2. Navigate into the project
cd Rental_website

# 3. Install dependencies
npm install

# 4. Configure your MySQL database
#    → Update DB credentials in config/db.js (or .env)

# 5. Start the server
npm start
```

> **⚠️ Note:** Configure your MySQL database credentials before running the app.

---

## 🗄️ Database Setup

1. Create a MySQL database (e.g., `rental_db`)
2. Import the provided SQL schema file
3. Update your connection settings in the config file
